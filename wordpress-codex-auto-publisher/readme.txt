=== WordPress Codex Auto Publisher ===
Contributors: rastar13
Version: 1.0
Description: Publishes content bundles generated by Codex Cloud.

== Installation ==
1. Upload the `wordpress-codex-auto-publisher` folder to your `wp-content/plugins/` directory.
2. Activate the plugin via the WordPress admin.
3. Configure the Codex API key in Settings â†’ Codex Auto Publisher.

== REST endpoint ==
POST /wp-json/codex/v2/publish
Requires header: Codex-Key: <your-key>

== Behavior ==
On receiving a valid publish packet this plugin will create a post using the fields provided in the incoming bundle (title, content_html, status, tags, categories). Logs are written to wp-content/uploads/codex-auto-publisher/log.txt
